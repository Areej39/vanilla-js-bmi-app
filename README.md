# BMI Calculator

A simple and responsive BMI (Body Mass Index) Calculator built using HTML, CSS, and JavaScript. This application allows users to calculate their BMI based on weight and height inputs and displays the corresponding health category.

## Features

* Calculate BMI using weight (kg) and height (cm)
* Displays BMI value with one decimal precision
* Shows BMI category:

  * Underweight
  * Normal
  * Overweight
  * Obese
* Input validation for empty, invalid, and negative values
* Reset button to clear inputs and results
* Enter key support for quick calculation
* Responsive design for desktop and mobile devices
* Dynamic category styling based on BMI result

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## BMI Categories

| Category    | BMI Range      |
| ----------- | -------------- |
| Underweight | Less than 18.5 |
| Normal      | 18.5 - 24.9    |
| Overweight  | 25.0 - 29.9    |
| Obese       | 30.0 and above |

## How to Use

1. Enter your weight in kilograms.
2. Enter your height in centimeters.
3. Click the **Calculate BMI** button or press **Enter**.
4. View your BMI value and category.
5. Click **Reset** to clear all fields and results.
